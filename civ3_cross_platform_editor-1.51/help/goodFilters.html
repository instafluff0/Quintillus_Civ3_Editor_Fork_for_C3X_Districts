Resource Filters

Some filters support an equals operand, some support greater than, less than, and equals operand (referred to as GT, LT, EQ; symbols >, <, and =), and the resource name filter does not require an operand. If a version is indicated in parentheses, the filter was added in that version and will not be available if you are on an older version. Resource filters were first introduced in version 1.33.

The available resource filters are:

  • name: search for resources whose name includes the text entered. Does not require an operand
  • type: enter "strategic", "luxury", or "bonus" to filter to that type of resource
  • prerequisite: search for resources that require a particular technology. Accepts any technology name, or "None" for resources that don't require any technologies
  • era: allows filtering to resources that become available in certain eras. Accepts both era numbers (0 through 4, where 0 is no era), and era names (e.g. None, "Middle Ages").
  • food: filters based on the food bonus
  • shields, also available as production: filters based on the shield/production bonus
  • commerce: filters based on the commerce bonus
  • appearance, also available as appearanceRatio: filters based on the appearance ratio
  • disappearance, also available as disappearanceRatio: filters based on the disappearance ratio
  • icon: filters based on the resource's icon index. most likely useful for finding out-of-bounds indices
