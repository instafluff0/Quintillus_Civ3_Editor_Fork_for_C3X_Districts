Input from SAV

The editor has some support for inputting BIQ data from a .SAV file (a Conquests savegame). There are some recommendations about this SAV:

  • The SAV should be for the latest version of Conquests (it may work with pre-Conquests 1.22). PTW and Vanilla are not supported.
  • Only the retail version of Conquests was tested, though others will probably work.

The SAV may be either compressed or uncompressed.

Only the custom rules are extracted at this time; custom player and map data is not extracted.

The primary envisioned use for this is to recover BIQ files for scenarios for which you no longer have the original scenario, but do have some old savegames.

To Use

Simply click the Input From SAV button upon starting the program, and then choose the SAV you want. Make any modifications you like, and then save it as a BIQ.

Future Possibilities

The capabilities of this functionality could potentially be expanded to allow extracting maps or custom player data. If you want this functionality, please request it in the forum thread - so far little has been heard about this ability.
